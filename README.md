###克隆该项目至本地
```
git clone https://github.com/RangelZZZ/reflux-demo
```
* 执行 `npm install`
###执行第一个 Demo（Reflux+React 实现页面渲染`hello world！`）

* 执行`npm run webpack`
* 运行`hello.html`

###执行第二个 Demo（Reflux+React 实现页面显示用户输入的文本内容）
* 运行`hello-text.html`

###执行第三个 Demo（Reflux+React+Express 实现页面渲染`hello world！`）
* 执行`npm start`
* 访问 `localhost:3000`
